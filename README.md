# AstroHack Interactive Platform

A responsive frontend web application built as part of a Frontend Interview Assignment.
The project focuses on interactive UI design, smooth animations, and accessibility.

## Features

### Core Features

* Animated hero section with glowing background behind astronaut
* Interactive timeline slider
* Image gallery with hover effects (zoom and glow)
* Modern dark-themed UI

### Performance and UX

* Lazy loading for images
* Smooth scroll-based animations
* Optimized rendering using React hooks

### Accessibility

* Semantic HTML structure (`main`, `section`, `footer`)
* Keyboard navigation support
* Visible focus states
* ARIA attributes for screen readers

### Responsiveness

* Fully responsive across mobile, tablet, and desktop
* Works on both high-density (Retina) and standard screens
